EMoov

Sistema de mobilidade interna e controle de frota desenvolvido como TCC, com foco no gerenciamento de veículos, motoristas, funcionários e corridas.

O Desafio

Organizar o fluxo de solicitações de corridas e o controle de frota em um ambiente digital, facilitando a gestão de veículos, motoristas e usuários envolvidos no processo de mobilidade interna.

Tecnologias

NestJS Next.js React Native Expo TypeScript Prisma ORM PostgreSQL Supabase JWT Socket.IO

A Solução

O EMoov foi estruturado como uma solução completa para cadastro, controle e acompanhamento de corridas, combinando back-end, regras de negócio, banco de dados e app mobile.

  • Gerenciamento de veículos, motoristas, funcionários e solicitações.
  • Autenticação com JWT para controle de acesso.
  • Banco de dados PostgreSQL com Prisma ORM.
  • Comunicação em tempo real com Socket.IO.
  • Back-end com NestJS, banco de dados PostgreSQL e aplicação mobile com React Native e Expo.

Observação

Este projeto está em processo de comercialização e, por esse motivo, seu código-fonte não está disponível publicamente. As informações apresentadas representam apenas uma visão geral da solução desenvolvida.
