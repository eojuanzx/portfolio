EMoov Quality Assurance

Projeto de testes manuais da API REST do sistema EMoov, com foco em planejamento, execução, evidências e validação funcional de endpoints.

O Desafio

Validar endpoints da API REST do EMoov por meio de testes manuais, garantindo organização dos cenários, rastreabilidade das evidências e atenção à qualidade das respostas retornadas.

Tecnologias

NestJS TypeScript Prisma ORM PostgreSQL JWT Thunder Client QA

A Solução

Foram organizados casos de teste para validar fluxos da API, respostas esperadas, autenticação, regras de negócio e comportamento dos principais módulos do sistema.

  • Planejamento dos testes manuais de API.
  • Criação e execução de casos de teste.
  • Registro de evidências dos testes realizados.
  • Validação funcional de endpoints com Thunder Client.
